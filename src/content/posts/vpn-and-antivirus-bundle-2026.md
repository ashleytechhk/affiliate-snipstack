## Troubleshooting

**Antivirus scan is slow on first run**
Normal. The initial scan indexes every file. After that, only new or changed files are checked. Let it finish overnight if needed.

**VPN slows down streaming**
Switch from the auto-selected server to one geographically closer. On the server list, sort by latency. WireGuard protocol is fastest — select it under Settings → VPN Protocol.

**App conflicts with old antivirus**
If you still have Norton or McAfee installed, uninstall it first. Two antivirus engines running simultaneously cause performance issues and false positives. Use the dedicated removal tool from the old vendor's site, not just the standard uninstaller.

**Family member keeps getting disconnected**
Check that their device has battery-saver mode set to allow Surfshark in the background. On Android: Settings → Battery → App Battery Management → Surfshark → No restrictions.
